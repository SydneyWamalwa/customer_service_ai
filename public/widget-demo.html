Welcome to the Customer Service AI Widget demo! This page demonstrates how easy it is to integrate our AI-powered customer support into your website.

Features

  • AI-powered responses using company-specific knowledge
  • Customizable branding (colors, logo, agent name)
  • Anonymous and authenticated customer support
  • Seamless escalation to human agents
  • Mobile-responsive design
  • Easy integration with just a few lines of code

How to Integrate

Add the following code snippet to your website before the closing </body> tag:

<script>
  window.customerAIConfig = {
    companyId: 'YOUR_COMPANY_ID',
    apiUrl: 'https://your-worker-url.workers.dev',
    user: {
      id: 'USER_ID',      // Optional
      name: 'USER_NAME',  // Optional
      email: 'USER_EMAIL' // Optional
    }
  };
</script>
<script src="https://your-worker-url.workers.dev/widget.js" async></script>
Note: Replace YOUR_COMPANY_ID with your actual company ID from the admin dashboard. The user information is optional and can be omitted for anonymous support.

Try It Now!
